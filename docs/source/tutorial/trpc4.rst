TRPC4 tutorial data set
***********************

To demonstrate how TranSPHIRE works, a TRPC4 tutorial test data set can be downloaded here: |DOWNLOAD_PATH|

Untar the downloaded file ``transphire_trpc4_tutorial.tar.gz``:

>>> tar -xf transphire_trpc4_data_set.tar.gz

This extracts a folder called ``transphire_trpc4_tutorial``.
