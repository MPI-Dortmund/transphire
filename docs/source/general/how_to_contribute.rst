How to contribute
=================


There are several ways how one can contribute to the TranSPHIRE project.

    - Bug reports
    - Feature requests
    - Missing documentation
    - Nice words

If you are interested to contribute, feel free to write an e-mail to markus.stabrin@mpi-dortmund.mpg.de.
